本说明用于统一客服、投诉、账号注销、数据访问、删除和导出请求的入口与处理口径。

## 1. 客服入口

- 客服电话：`【待确认：客服电话】`
- 客服邮箱：`【待确认：客服邮箱】`
- 在线客服入口：`【待确认：在线客服入口】`
- 服务时间：`【待确认：客服服务时间】`

客服可处理：

- 登录问题。
- API Key 创建、停用和使用问题。
- 套餐、订单、支付和退款问题。
- 发票申请问题。
- 用量统计和调用日志问题。
- 内容安全误判申诉。

客服不会索要：

- API Key 明文。
- 微信支付密码。
- 银行卡密码。
- 短信验证码。
- Cookie。
- 商户私钥、API v3 Key 或 AppSecret。

## 2. 投诉入口

- 投诉邮箱或入口：`【待确认：投诉邮箱或投诉入口】`
- 投诉处理时限：`【待确认：投诉处理时限】`

可投诉事项：

- 订单、退款、发票争议。
- 内容安全误判或处置异议。
- 个人信息处理问题。
- 客服处理不当。
- 疑似账号盗用、API Key 滥用或安全事件。

建议提供：

- 账号标识。
- 订单号或请求 ID。
- 问题发生时间。
- 问题描述。
- 截图或其他证明材料。

不要提供密码、验证码、完整银行卡号或完整 API Key。

## 3. 账号注销

用户可以申请注销账号。注销前应确认：

- 没有未完成订单、退款或发票申请。
- 已保存需要留存的用量、订单和发票记录。
- 已停用不再使用的 API Key。
- 注销后账号、套餐、API Key 和部分记录可能无法恢复。

注销处理时限：`【待确认：账号注销处理时限】`

## 4. 个人信息请求

用户可以申请：

- 访问或复制个人信息。
- 更正或补充不准确的信息。
- 删除个人信息。
- 导出个人信息。
- 撤回授权或同意。
- 解释个人信息处理规则。

数据请求入口：

- 客服邮箱：`【待确认：客服邮箱】`
- 投诉入口：`【待确认：投诉邮箱或投诉入口】`
- 数据导出方式：`【待确认：数据导出方式】`

平台在处理请求前可能需要核验用户身份和请求范围。

## 5. 安全事件报告

如果用户发现以下问题，应尽快联系平台：

- API Key 泄露或被盗用。
- 账号异常登录。
- 订单或支付状态异常。
- 疑似数据泄露。
- 疑似模型输出违法或侵权内容。
